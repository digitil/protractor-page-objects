Parameters:
Name Type Description
pages Array.<Page~PageDefinition>

Examples

Pass in a variable number of pages

var app = new PageObjects(login, home, settings);

Pass in an array of pages

var app = new PageObjects([login, home, settings]);

Or mix the two

var app = new PageObjects(login, [home], settings);
